Models

All hand made by yours truly

(These models can be converted to other formats, but are made for Minecraft)

(I would suggest converting or making changes to them in BlockBench for free)

Each download contains a .bbmodel file, a .json file, and a .json.textures file, as well as all the textures. Use the .json.textures file to map the textures correctly, if they aren't mapped already

Please note that newer models do not include a .json.textures file, as there is only one texture to be used on the model, meaning it is a lot simpler to map the texture back onto the model.
